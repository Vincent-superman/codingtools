 

  DevTools 是一款纯前端开发者工具集，所有功能均在浏览器本地运行，零依赖后端，数据不外传。项目采用原生 HTML / CSS / JavaScript 构建，无需打包构建，单页面承载约 30 项常用工具，覆盖
  JSON / YAML / XML / SQL 格式化与互转、JSON 内容比对、Base64 文本与图片编解码、URL / HTML 实体 / Unicode 编解码、AES 加解密、二维码生成与解析、UUID / Hash / 强密码 / Lorem
  占位文本生成、正则测试、文本差异、Markdown 预览、大小写与进制转换、时间戳与 Cron 解析、颜色与 CSS 渐变、JWT 解码、HTTP 状态码速查等。界面采用 GitHub
  风格深色主题，左侧分组导航、顶部支持中英文切换，移动端自适应。轻量、即开即用、隐私友好，适合作为日常工作的常驻工具页。

   

  DevTools is a browser-only developer toolkit — every feature runs locally with no backend and zero data upload. Built with vanilla HTML, CSS, and JavaScript, it ships as a
  single page hosting roughly 30 everyday utilities: JSON / YAML / XML / SQL formatting and conversion, JSON content diff, Base64 text and image codecs, URL / HTML entity /
  Unicode encoding, AES encrypt and decrypt, QR code generation and decoding, UUID / hash / strong password / Lorem ipsum generators, regex tester, text diff, Markdown preview,
  case and number-base conversion, timestamp and Cron parsing, color and CSS gradient, JWT decoder, and HTTP status reference. A GitHub-style dark theme pairs with a grouped
  sidebar, a top-right ZH / EN switcher, and responsive mobile layout. It's lightweight, instant, and privacy-friendly — a practical daily companion you can pin in any browser
  tab.
